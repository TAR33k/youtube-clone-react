# YouTube Clone React

A modern YouTube clone built with React, featuring video browsing, user profiles, and responsive design. This project demonstrates a full-stack frontend implementation with routing, state management, and component-based architecture.

Data fetched from the Youtube Data API v3.

## Features

- **Video Browsing**: Browse videos with categories and recommendations
- **Responsive Design**: Optimized for desktop and mobile devices
- **Modern UI**: Clean, YouTube-inspired interface

## Tech Stack

- **Frontend Framework**: React 19
- **Build Tool**: Vite

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/TAR33k/youtube-clone-react.git
   cd youtube-clone-react
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the development server**

   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
